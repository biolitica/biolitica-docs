# Biolítica Docs (Capa 1 transversal)

Repositorio oficial para documentación transversal y pública de Biolítica.

## Qué vive aquí

- **Master WBS única** para coordinación cross-repo.
- **Source of Truth (SoT)** por tema/documento.
- **Gobernanza documental** para evitar divergencia.
- **Índice de enlaces** hacia documentación de repos de componentes.

## Cómo usar este sitio

1. Revisa **Master WBS** para prioridades y estado transversal.
2. Consulta **Source of Truth** antes de editar specs/contratos/runbooks.
3. Usa **Repo Links** para ir al detalle técnico en cada repo.
4. Sigue **Governance** para proponer cambios sin duplicar información.

## Principios operativos

- Este repo define el marco transversal; los repos de componentes mantienen el detalle de implementación.
- No se duplican contratos/specs: se enlaza al documento SoT.
- Si un path no está validado todavía, se marca explícitamente como `TODO: verify path`.
